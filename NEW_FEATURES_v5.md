## 📦 新增脚本快速参考

### 13-StorageOptimize.ps1 - 存储优化 ⭐⭐⭐⭐

**一句话:** SSD 性能优化 + 磁盘清理

**何时使用:**
- ✅ 系统盘空间不足
- ✅ SSD 用户想要最佳性能
- ✅ 系统运行缓慢
- ✅ 临时文件积累

**执行时间:** 2 分钟  
**需要重启:** 否

**主要功能:**
```
✓ SSD/HDD 检测
✓ TRIM 启用
✓ SysMain 服务配置建议
✓ 临时文件清理
✓ Windows Update 缓存清理
✓ 页面文件配置检查
✓ 磁盘空间分析
```

**预期效果:**
- 释放 1-5GB 空间
- SSD 性能提升 5-15%
- 减少后台磁盘活动

---

### 14-PrivacyOptimize.ps1 - 隐私优化 ⭐⭐⭐⭐⭐

**一句话:** 禁用遥测 + 隐私保护

**何时使用:**
- ✅ 关注隐私保护
- ✅ 减少数据上传
- ✅ 禁用广告追踪
- ✅ 新系统初始化

**执行时间:** 1 分钟  
**需要重启:** 建议

**主要功能:**
```
✓ 遥测服务禁用 (DiagTrack, dmwappushservice)
✓ 诊断数据收集禁用
✓ 广告 ID 禁用
✓ Cortana 禁用
✓ 位置追踪禁用
✓ Windows Ink 禁用
✓ 活动历史禁用
✓ 反馈通知禁用
✓ Bing 搜索禁用
✓ 后台应用禁用
```

**预期效果:**
- 减少 80-95% 遥测数据
- 减少后台资源占用
- 提升隐私保护级别

**注意:** 需要输入 "YES" 确认执行

---

```
✓ 启动项分析

✓ 遥测服务自动禁用

```

**预期效果:**

## ⚠️ 重要提示

### 隐私优化 (14-PrivacyOptimize.ps1)

**可能影响的功能:**
- ❌ Cortana 语音助手
- ❌ Windows 时间线
- ❌ 跨设备同步
- ❌ Windows Ink (手写笔)

**建议:** 不使用这些功能的用户可安全禁用

---

### 启动优化 (15-StartupOptimize.ps1)

**不要禁用:**
- ✅ 杀毒软件
- ✅ 硬件驱动工具
- ✅ 输入法

**可以禁用:**
- ✅ 游戏平台 (Steam, Epic)
- ✅ 通讯软件 (Discord, Skype)
- ✅ 浏览器更新程序

---

### 存储优化 (13-StorageOptimize.ps1)

**SSD 用户:**
- ✅ 禁用 SysMain/Superfetch
- ✅ 启用 TRIM (自动)
- ✅ 保持 20%+ 可用空间

**HDD 用户:**
- ✅ 保留 SysMain/Superfetch
- ✅ 定期碎片整理

---

### 更新管理 (16-UpdateOptimize.ps1)

**推荐设置:**
- ✅ 安全更新：自动安装
- ✅ 功能更新：延迟 2-3 个月
- ✅ 驱动更新：从官网下载
- ✅ P2P 分发：禁用
